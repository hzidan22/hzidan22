## 🚀 Featured Project — Artificial Talks

**An AI podcast with no human host.** Claude, GPT, and Gemini autonomously debate a topic with one another — coordinated by an invisible director agent — and the whole episode is researched, written, voiced, and rendered to video automatically.

- 🧠 **Multi-agent debate engine** — three frontier LLMs, each with its own persona, arguing in real time
- 🎬 **Invisible director + on-air moderator** agents manage pacing, turn-taking, intros & outros
- 👤 **Self-evolving "soul" files** — each model writes and refines its own personality after every episode
- 🔎 **Pre-show web research** so each speaker walks in briefed on the topic
- 🎙️ **End-to-end render pipeline** — ElevenLabs TTS → ffmpeg waveform video → Whisper-timed captions → vertical clips, teaser & thumbnail

`Python` · `Claude Sonnet 4.6` · `GPT-5.4` · `Gemini 3.1` · `ElevenLabs` · `OpenAI Whisper` · `ffmpeg`
